# VizagProperty - Visakhapatnam Real Estate Platform

A modern, AI-powered real estate platform for Visakhapatnam with voice search, chatbot assistance, and seamless user experience.

## Features

### Core Features
- **AI-Powered Search** - Natural language search that understands what you want
- **Voice Assistant** - Hands-free property search using voice commands
- **Smart Chatbot** - 24/7 AI assistant to help find properties and answer questions
- **Instant Results** - Lightning-fast property search and filtering
- **WhatsApp Integration** - One-tap contact with property agents

### User Features
- **User Authentication** - Secure sign up/sign in with email and password
- **User Profiles** - Buyer, Seller, or Agent profiles
- **Favorites** - Save properties for later viewing
- **Property Views** - Track property views and analytics
- **Search History** - Keep track of your searches

### Property Features
- **Multiple Property Types** - Apartments, Villas, Plots, Commercial, Penthouses, Farmhouses
- **Detailed Listings** - Comprehensive property information with images
- **Advanced Filters** - Filter by price, location, bedrooms, type, and more
- **Verified Listings** - Admin-verified properties
- **Featured Properties** - Premium property highlights

### Seller/Agent Features
- **List Properties** - Easy-to-use property listing form
- **Manage Listings** - Update and manage your properties
- **Contact Management** - Direct phone and WhatsApp integration

## Technology Stack

- **Frontend**: React 18 + TypeScript
- **Styling**: Tailwind CSS
- **Routing**: React Router v6
- **Database**: Supabase (PostgreSQL)
- **Authentication**: Supabase Auth
- **Serverless Functions**: Supabase Edge Functions
- **Voice Recognition**: Web Speech API
- **Build Tool**: Vite

## Getting Started

The application is ready to use! Simply open the app and:

1. **Browse Properties** - Start exploring properties on the homepage
2. **Use Voice Search** - Click the microphone icon and speak your search
3. **Chat with AI** - Click the chat bubble in the bottom right to ask questions
4. **Sign Up** - Create an account to save favorites and list properties
5. **List Properties** - Sellers and agents can add new properties
